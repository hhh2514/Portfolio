Introduction

Upgrading the equipment directly connected to the agents allows for recognizing who is logged into the workstation, thus re-establishing the connection between GLPI and the Pulse Secure application, which enables background installations on different workstations.

Application deployment includes updating applications already present on the agents' workstations or installing applications necessary for the agents' functionality.

Furthermore, deployment is done remotely, allowing for increased simplicity and convenience.

All of this is done through an application called "Pulse Secure."
